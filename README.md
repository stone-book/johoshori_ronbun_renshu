# 概要

情報処理技術者試験の高度試験区分にある、午後2の論述試験の練習用のためのワードファイル版の原稿用紙です。
「ronbun_format.docx」が原稿用紙です。
論文の練習にお使いいただければと思います。

## 作った方法
レイアウト -> ページ設定 にある「文字数と行数の指定」で、横25文字、縦32行に設定しております。
同上の設定画面にある、グリッド線を表示し、それを参考にしながら、ヘッダー/フッターモードで地味にオートシェイプ図形の線を張り付けました。
（ヘッダー/フッターモードだと本文に影響がない。もっとスマートな方法があるかもしれません。）
